

## quant-ML

A unified framework for quantitative machine learning research.

This repository explores machine learning from a quantitative perspective, focusing on learning dynamics, optimization, and functional representations such as backpropagation and functional gradient methods.

---

## Scope

- Quantitative ML Foundations:
  Learning theory, optimization, and statistical structure of ML models

- Backpropagation & Learning Dynamics:
  Gradient-based learning, error propagation, and training stability

- Functional Learning Methods:
  Functional gradient approaches and continuous-view formulations of learning

- Model Analysis:
  Understanding behavior, convergence, and representation in ML systems
